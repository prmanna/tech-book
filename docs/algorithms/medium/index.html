      Hashing #

      • Two Sum: Find a pair in array whose sum equals to the target input: [10,4,1,3,2] Target: 7 Output: [4,3]

        Hint - insert in hash and then start searching from first element, find the difference with the target and find the hash.

      • Logest consecutive sequence: find length of longest consecutive sequence from given array input: [10, 4, 20, 1,3,2] Output: [1,2,3,4]
